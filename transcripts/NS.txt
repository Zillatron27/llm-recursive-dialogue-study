Since you're pulling the Windows drive out first (smart move to avoid any confusion), here's the straightforward approach:

The clean way using Clonezilla:

    Boot into a Clonezilla live USB (grab the ISO from clonezilla.org if you don't have it)
    Once booted, select:
        device-device mode
        Beginner mode is fine
        disk_to_local_disk
    Select your current Bazzite drive as source
    Select the new 4TB as destination
    Let it do its thing

Important considerations:

    After cloning, you'll want to expand the partitions to use that full 4TB. Clonezilla will copy the exact partition layout, so you'll initially have the same size partitions with tons of unallocated space
    You can expand the filesystem after booting back into Bazzite using gnome-disks (GUI) or growpart + filesystem resize commands
    Make sure your BIOS/UEFI boot order points to the new drive after the clone

Alternative if you want to stay in Bazzite:

You could also use dd or rsync from within a live USB of Bazzite itself, but Clonezilla is more foolproof for a full disk clone with all the bootloader stuff.
